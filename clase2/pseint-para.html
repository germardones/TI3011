Para (For)

La estructura Para es ideal cuando se conoce de antemano el número exacto de iteraciones. Integra el contador, la condición de parada y el incremento/paso en una sola línea de código.

pseint - Para.psc 
Algoritmo BuclePara
  
  Definir i Como Entero;
  
  Para i<-1 Hasta 3 Con Paso 1 Hacer
    Escribir i;
  FinPara

  Escribir "Adiós";

FinAlgoritmo
