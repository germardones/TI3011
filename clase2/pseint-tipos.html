Tipos de Datos Primitivos

Pasa el mouse sobre el código o la memoria para entender las diferencias entre Entero, Real, Caracter y Lógico.

pseint - TiposBasicos.psc 
Algoritmo TiposBasicos
  
  // Fase 1: Creación de variables en memoria
  Definir edad Como Entero;
  Definir precio Como Real;
  Definir letra Como Caracter;
  Definir esNoche Como Logico;
  
  // Fase 2: Inyectar datos en las variables
  edad <- 25;
  precio <- 19.99;
  letra <- 'A';
  esNoche <- Verdadero;

FinAlgoritmo

Tipos de Variables

Entero

Números sin decimales.
Ej: 25, -10

Real

Números fraccionarios.
Ej: 19.99, 3.14

Caracter

Símbolo, letra o texto.
Ej: 'A', "Hola"

Lógico

Estados booleanos de verdad.
Ej: Verdadero, Falso
