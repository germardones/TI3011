¿Qué es GitHub Copilot?

GitHub Copilot es un asistente de programación con IA desarrollado por GitHub en colaboración con OpenAI. Usa modelos como GPT-4 (Codex) para sugerir líneas y funciones completas mientras escribes código.

Cómo funciona

El modelo fue entrenado con millones de repositorios públicos de GitHub. Cuando escribes, envía el contexto de tu código a la API y devuelve sugerencias.

Buenas prácticas

✅

Revisar siempre el código sugerido antes de aceptar

Copilot puede sugerir código que funciona pero que contiene vulnerabilidades. Cada línea que aceptas es tu responsabilidad.

✅

Usar comentarios descriptivos para guiar la sugerencia

Un comentario claro como # función que valida email y retorna True/False produce mejores sugerencias.

✅

Verificar que no se incluya información sensible en el contexto

El contexto del archivo activo se envía a la API. Nunca trabajes con archivos de configuración con contraseñas reales mientras usas Copilot.

✅

Entender el código antes de usarlo en producción

Si no puedes explicar qué hace el código sugerido, no lo uses. La IA puede producir código correcto en apariencia pero incorrecto en comportamiento.

Limitaciones y riesgos

⚠ Vulnerabilidades conocidas — Puede sugerir código con patrones inseguros aprendidos de repositorios públicos con malas prácticas.

⚠ No garantiza corrección — El código puede compilar y ejecutarse sin errores, pero producir resultados incorrectos en casos borde.

⚠ Reproduce patrones inseguros — Si el proyecto ya tiene código inseguro, Copilot puede seguir el mismo patrón. Garbage in, garbage out.

⚠ No reemplaza al programador — Es un asistente, no un desarrollador. La comprensión, el diseño y la responsabilidad del código siguen siendo tuyas.

Ejemplo práctico

Copilot convierte comentarios descriptivos en código. Así funciona el flujo comentario → sugerencia:

🤖 copilot_demo.py — flujo comentario → sugerencia
# Función que valida un RUT chileno # Retorna True si es válido, False si no
def validar_rut(rut: str) -> bool: """Valida un RUT chileno con dígito verificador.""" rut = rut.replace("-", "").replace(".", "").upper() if len(rut) < 2: return False cuerpo, dv = rut[:-1], rut[-1] suma, mult = 0, 2 for d in reversed(cuerpo): suma += int(d) * mult mult = 2 if mult == 7 else mult + 1 esperado = str(11 - (suma % 11)).replace("10","K").replace("11","0") return dv == esperado

✅ Qué hacer después de recibir una sugerencia: Lee el código, entiende la lógica del módulo, pruébalo con casos de prueba (RUTs válidos e inválidos), y solo entonces acepta o adapta la función.
