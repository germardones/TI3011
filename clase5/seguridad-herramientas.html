Bandit — Análisis estático para Python

Bandit es una herramienta de análisis estático de seguridad desarrollada por PyCQA. Escanea código Python en busca de patrones de seguridad comunes como inyección de comandos, deserialización insegura y contraseñas hardcodeadas.

Instalación:

pip install bandit

Uso:

bandit archivo.py
bandit -r mi_proyecto/

Código que Bandit detectaría como inseguro:

# ❌ INSEGURO — Bandit detecta estos problemas
import subprocess

# B602: subprocess con shell=True — riesgo de inyección
comando = input("Ingresa comando: ")
subprocess.call(comando, shell=True)

import pickle
# B301: deserialización insegura
datos = pickle.loads(entrada_usuario)

# B105: contraseña hardcodeada
password = "admin123"

import hashlib
# B324: uso de MD5 (algoritmo débil)
h = hashlib.md5(dato.encode()).hexdigest()

Versiones corregidas:

# ✅ SEGURO — versiones corregidas
import subprocess

# Sin shell=True, argumentos como lista
subprocess.run(["ls", "-la"], capture_output=True)

# Usar json en lugar de pickle para datos externos
import json
datos = json.loads(entrada_usuario)

# Cargar contraseña desde variable de entorno
import os
password = os.environ.get("DB_PASSWORD")

# Usar SHA-256 en lugar de MD5
import hashlib
h = hashlib.sha256(dato.encode()).hexdigest()

SonarQube — Plataforma de calidad de código

SonarQube es una plataforma web para inspección continua de calidad de código. Detecta bugs, vulnerabilidades, code smells y mide cobertura de tests. Soporta más de 30 lenguajes incluyendo Python.

# sonar-project.properties sonar.projectKey=mi-proyecto-python sonar.projectName=Mi Proyecto Python sonar.sources=src sonar.python.version=3.11 sonar.host.url=http://localhost:9000
# Ejecutar análisis sonar-scanner
