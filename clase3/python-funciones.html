Tipos de funciones

Sin parámetros ni retorno

Ejecuta una acción simple, sin recibir ni devolver datos.

def saludar():
    print("¡Hola, Python!")

# Llamada:
saludar()  # No retorna nada
¡Hola, Python!

Con parámetros, sin retorno

Recibe datos via parámetros y los usa internamente.

def saludar_persona(nombre, edad):
    print(f"Hola {nombre}, tienes {edad} años")

# Llamada con argumentos:
saludar_persona("Ana", 22)
Hola Ana, tienes 22 años

Con parámetros y retorno

Recibe datos, los procesa y devuelve un resultado.

def potencia(base, exp):
    resultado = base ** exp
    return resultado

r = potencia(2, 10)
print(r)
1024

Parámetros por defecto

Los parámetros puede tener un valor predeterminado opcional.

def conectar(host, puerto=80):
    print(f"Conectando a {host}:{puerto}")

conectar("google.com")         # usa 80
conectar("google.com", 443)    # usa 443
Conectando a google.com:80
Conectando a google.com:443

Múltiples valores de retorno

Python puede retornar varios valores como tupla.

def min_max(lista):
    return min(lista), max(lista)

minimo, maximo = min_max([3, 1, 9, 5])
print(minimo, maximo)
1   9

Parámetros nombrados (kwargs)

Puedes pasar argumentos por nombre en cualquier orden.

def crear_perfil(nombre, edad, ciudad):
    return f"{nombre}, {edad}, {ciudad}"

# Orden no importa con kwargs:
crear_perfil(ciudad="Iquique", nombre="Ana", edad=22)
Ana, 22, Iquique
