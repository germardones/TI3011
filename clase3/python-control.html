🔀 Condicional if

Pasa el cursor sobre cada línea de código para ver el nodo correspondiente en el diagrama.

condicional.py
nota = int(input("Nota (0-100): "))

if nota >= 90:
    print("Sobresaliente ")
elif nota >= 70:
    print("Aprobado ")
elif nota >= 55:
    print("Recuperación ️")
else:
    print("Reprobado ")

🔁 Bucle for

Pasa el cursor sobre cada línea para ver el diagrama de flujo correspondiente.

bucle_for.py
for i in range(1, 6):
    print(f"Iteración {i}")

# range(inicio, fin, paso)
# ↳ inicio: valor inicial (incluido)
# ↳ fin: valor final (excluido)
# ↳ paso: incremento (defecto 1)

🔄 Bucle while

Pasa el cursor sobre cada línea para ver el diagrama de flujo correspondiente.

bucle_while.py
contador = 1

while contador <= 5:
    print(f"Vuelta {contador}")
    contador += 1

print("Bucle terminado")
