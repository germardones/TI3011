🔀 Condicional if / elif / else

Ejecuta un bloque de código u otro según qué condición sea verdadera. Solo el primer bloque cuya condición sea True se ejecuta.

Sintaxis
if condición: # bloque verdadero elif otra_condición: # bloque alternativo (opcional) else: # bloque por defecto (opcional)
  • Solo un bloque se ejecuta: el primero cuya condición sea True.
  • elif y else son opcionales; puede haber varios elif.
  • El indentado (4 espacios) define qué líneas pertenecen al bloque.
condicional.py
1nota = int(input("Nota (0-100): "))
2 
3if nota >= 90:
4print("Sobresaliente")
5elif nota >= 70:
6print("Aprobado")
7elif nota >= 55:
8print("Recuperación")
9else:
10print("Reprobado")

🔁 Bucle for

Recorre cada elemento de una secuencia y ejecuta el bloque una vez por cada elemento. Ideal cuando sabes de antemano cuántas veces necesitas iterar.

Sintaxis
for variable in secuencia: # bloque que se repite
  • range(inicio, fin) genera números desde inicio hasta fin - 1.
  • La variable de iteración toma automáticamente cada valor de la secuencia.
  • No necesitas actualizar la variable manualmente: el for lo hace solo.
bucle_for.py
1for i in range(1, 6):
2print(f"Iteración {i}")
3 
4# range(inicio, fin, paso)
5# ↳ inicio: valor inicial (incluido)
6# ↳ fin: valor final (excluido)
7# ↳ paso: incremento (por defecto 1)

🔄 Bucle while

Repite un bloque mientras una condición sea True. A diferencia de for, no recorre una secuencia fija: tú controlas cuándo termina.

Sintaxis
while condición: # bloque que se repite # (actualiza la condición aquí)
  • La condición se evalúa antes de cada iteración. Si es falsa desde el inicio, el bloque nunca se ejecuta.
  • Es tu responsabilidad actualizar la variable de control para que el bucle termine.
  • Si la condición nunca se vuelve False, el bucle es infinito.
bucle_while.py
1contador = 1
2 
3while contador <= 5:
4print(f"Vuelta {contador}")
5contador += 1
6 
7print("Bucle terminado")

🔃 do-while en Python

Python no tiene do-while nativo. Se simula con while True + break. El cuerpo siempre se ejecuta al menos una vez, sin importar la condición.

Sintaxis
while True: # cuerpo (siempre ejecuta) if condición_de_salida: break
  • El cuerpo se ejecuta antes de verificar la condición: garantiza al menos una iteración.
  • break interrumpe el bucle inmediatamente y salta al código siguiente.
  • La condición de salida se evalúa dentro del cuerpo, no en el encabezado del bucle.
do_while.py
1while True:
2respuesta = input("¿Continuar? (s/n): ")
3if respuesta.lower() != 's':
4break
5 
6print("Programa finalizado")

🎯 match / case

Disponible desde Python 3.10. Equivalente al switch de otros lenguajes. Compara un valor contra múltiples patrones y ejecuta el bloque del primer caso que coincida.

Sintaxis
match valor: case patrón1: # bloque 1 case patrón2: # bloque 2 case _: # bloque por defecto
  • Solo el primer case que coincide se ejecuta; los restantes se ignoran.
  • case _ es el comodín: captura cualquier valor no cubierto anteriormente.
  • Requiere Python 3.10 o superior. En versiones anteriores se usa if/elif/else.
match_case.py
1opcion = int(input("Elige opción (1-3): "))
2 
3match opcion:
4case 1:
5print("▶ Iniciando...")
6case 2:
7print("💾 Guardando...")
8case 3:
9print("🚪 Saliendo...")
10case _:
11print("❌ Opción inválida")
