🖨️ print() — mostrar datos en pantalla

print() es la función de salida de Python. Todo lo que pongas dentro se mostrará en la consola cuando el programa se ejecute.

Sintaxis
print(valor, valor2, ...)
salida.py 
# Imprimir texto (string)
print("Hola, mundo")

# Imprimir un número
print(42)

# Imprimir una variable
nombre = "Ana"
print(nombre)

# Imprimir varios valores separados por coma
print("Edad:", 20)
print("Hola", nombre, "!")
SALIDA EN CONSOLA
Hola, mundo
42
Ana
Edad: 20
Hola Ana !
Separador automático: Cuando pasas varios valores separados por coma, print() los une con un espacio por defecto. Puedes cambiarlo con sep="," si lo necesitas.
📌Texto: siempre entre comillas simples '...' o dobles "..."
📌Números: se escriben directamente, sin comillas
📌Variables: se escriben por su nombre, sin comillas

✏️ print(f"") — f-strings

Un f-string (string formateado) te permite insertar el valor de una variable directamente dentro de un texto, usando llaves {}. Es la forma más moderna y legible de formatear salidas en Python.

Sintaxis
print(f"texto {variable} más texto")
fstrings.py 
nombre = "Carlos"
edad   = 22
nota   = 6.5

# Sin f-string (menos legible)
print("Hola, " + nombre + ". Tienes " + str(edad) + " años.")

# Con f-string (más legible)
print(f"Hola, {nombre}. Tienes {edad} años.")

# Expresiones dentro de las llaves
print(f"Tu nota es {nota}. Aprobado: {nota >= 4.0}")

# Redondear con :.2f
precio = 1234.5678
print(f"Precio: ${precio:.2f}")
SALIDA EN CONSOLA
Hola, Carlos. Tienes 22 años.
Hola, Carlos. Tienes 22 años.
Tu nota es 6.5. Aprobado: True
Precio: $1234.57
La f va justo antes de las comillas: f"..." o f'...'. Dentro de las llaves {} puedes escribir cualquier expresión válida de Python: variables, operaciones, llamadas a funciones.
🔢{valor:.2f} → muestra el número con 2 decimales
📐{valor:10} → reserva un ancho de 10 caracteres (útil para tablas)
⚡{2 + 2} → las expresiones se evalúan directamente

⌨️ input() — recibir datos del usuario

input() detiene la ejecución del programa, muestra un mensaje opcional en pantalla y espera a que el usuario escriba algo y presione Enter. Lo que el usuario escribe se guarda como texto.

Sintaxis
variable = input("Mensaje para el usuario: ")
Programa Python
→
Muestra el mensaje
→
Usuario escribe
→
Se guarda en la variable
entrada.py 
# El mensaje dentro de input() aparece en pantalla
nombre = input("¿Cuál es tu nombre? ")

# Usar lo que ingresó el usuario
print(f"Hola, {nombre}. Bienvenido.")
EJECUCIÓN — lo amarillo lo escribe el usuario
¿Cuál es tu nombre? Pedro
Hola, Pedro. Bienvenido.
input() siempre devuelve texto (str). Aunque el usuario escriba 25, Python lo recibe como la cadena "25", no como el número 25. Esto es importante al hacer cálculos.
💬El texto dentro de input("...") es el prompt — lo que el usuario ve como pregunta
⏸️El programa se pausa hasta que el usuario presione Enter
📝El resultado siempre es de tipo str, sin importar qué se escriba

🔄 Convertir la entrada del usuario

Como input() retorna siempre texto, si necesitas un número debes convertirlo con int() o float(). Sin esto, las operaciones matemáticas fallarán.

conversion.py 
# ❌ Sin conversión — error al sumar
texto = input("Escribe un número: ")  # "5"
print(texto + 10)  # TypeError: no se puede sumar str + int

# ✅ Con int() — convierte a entero
edad = int(input("¿Cuántos años tienes? "))
print(f"En 10 años tendrás {edad + 10} años.")

# ✅ Con float() — convierte a decimal
nota = float(input("Ingresa tu nota: "))
print(f"Nota ingresada: {nota:.1f}")
EJECUCIÓN CORRECTA
¿Cuántos años tienes? 22
En 10 años tendrás 32 años.
Ingresa tu nota: 6.5
Nota ingresada: 6.5
Patrón habitual: envuelve el input() directamente dentro de int() o float() en la misma línea. Así obtienes el tipo correcto de inmediato.
Funciones de conversión más usadas:
int() Convierte a número entero → 22
float() Convierte a decimal → 6.5
str() Convierte a texto → "22"

🚀 Programa completo

Aquí todo se combina: el usuario ingresa datos con input(), el programa los procesa y muestra el resultado con print(f"").

calculadora_notas.py 
# Calculadora de promedio

# — Entrada: pedimos datos al usuario —
nombre = input("Nombre del estudiante: ")
nota1  = float(input("Nota 1 (de 7.0): "))
nota2  = float(input("Nota 2 (de 7.0): "))
nota3  = float(input("Nota 3 (de 7.0): "))

# — Procesamiento —
promedio  = (nota1 + nota2 + nota3) / 3
aprobado  = promedio >= 4.0
estado    = "Aprobado" if aprobado else "Reprobado"

# — Salida: mostramos los resultados —
print()
print("--- Informe de notas ---")
print(f"Estudiante : {nombre}")
print(f"Promedio   : {promedio:.2f}")
print(f"Estado     : {estado}")
EJECUCIÓN COMPLETA
Nombre del estudiante: Valentina
Nota 1 (de 7.0): 5.5
Nota 2 (de 7.0): 6.0
Nota 3 (de 7.0): 4.5
--- Informe de notas ---
Estudiante : Valentina
Promedio : 5.33
Estado : Aprobado
Estructura clásica de un programa: Entrada → Procesamiento → Salida. Separar estas tres etapas en el código hace que sea más fácil de leer y depurar.
⌨️Entrada — input() para texto, int(input()) y float(input()) para números
⚙️Procesamiento — operaciones y lógica con las variables
🖨️Salida — print(f"...") para mostrar resultados con formato
