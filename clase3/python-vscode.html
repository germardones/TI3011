🧩 Instalar extensiones Python

VS Code por sí solo no entiende Python. Necesitas instalar dos extensiones oficiales de Microsoft.

1️⃣Presiona Ctrl+Shift+X para abrir el panel de Extensiones
2️⃣Busca Python (de Microsoft) y haz clic en Instalar
3️⃣Busca también Pylance (de Microsoft) e instálalo
4️⃣Si VS Code pide recargar, haz clic en "Recargar ventana"
¿Para qué sirve Pylance? Mejora enormemente el autocompletado. Te muestra los parámetros de las funciones y detecta errores antes de ejecutar.
🔍 Panel de Extensiones — busca:
🐍
Python

Pylance

📁 Abrir carpeta de proyecto

Siempre trabaja con una carpeta, no abriendo archivos sueltos. VS Code entiende mejor tu proyecto así.

1️⃣Ve a Archivo → Abrir Carpeta (o Ctrl+K luego Ctrl+O)
2️⃣Selecciona o crea una carpeta, por ejemplo TI3011_Python
3️⃣Haz clic en "Seleccionar carpeta" — verás el Explorador con tu carpeta
Nombra las carpetas sin espacios ni tildes. Usa guiones o guiones bajos: mi_proyecto no mi proyecto. La terminal trabaja mejor así.

🐍 Seleccionar el intérprete Python

VS Code necesita saber qué instalación de Python usar para ejecutar tu código.

1️⃣Presiona Ctrl+Shift+P para abrir la Paleta de Comandos
2️⃣Escribe Python: Select Interpreter y presiona Enter
3️⃣Selecciona la versión de Python 3 que aparece en la lista
BARRA DE ESTADO (abajo a la derecha):
🐍 Python 3.12.0 64-bit
← Haz clic aquí para cambiar el intérprete
¿No aparece Python? Instala Python desde python.org y marca "Add Python to PATH" durante la instalación. Luego reinicia VS Code.

📄 Crear y escribir un script

Con la extensión instalada y el intérprete seleccionado, ya puedes escribir código.

1️⃣En el Explorador, haz clic en el ícono Nuevo Archivo o usa Ctrl+N
2️⃣Nómbralo con extensión .py, por ejemplo: ejercicio1.py
3️⃣Escribe tu código — el resaltado de colores y autocompletado aparecen automáticamente
ejercicio1.py 
# Módulo 3 — Ejercicio con funciones

def calcular_promedio(notas):
    return sum(notas) / len(notas)

mis_notas = [85, 90, 78, 92]
prom = calcular_promedio(mis_notas)
print(f"Promedio: {prom}")

▶️ Ejecutar el script

Tienes tres formas de ejecutar tu script Python:

▶Botón ▶ — Esquina superior derecha del editor. La forma más rápida.
🖱️Clic derecho en el editor → "Ejecutar archivo Python en la terminal"
⌨️Terminal (Ctrl+`) → escribe python ejercicio1.py
TERMINAL INTEGRADO
PS C:\TI3011_Python> python ejercicio1.py
Promedio: 86.25
PS C:\TI3011_Python> █
Tip: Guarda siempre el archivo (Ctrl+S) antes de ejecutar. Si hay un asterisco (*) en el nombre de la pestaña, el archivo tiene cambios sin guardar.

🔴 Leer los mensajes de error

Cuando tu código tiene un error, Python muestra un mensaje en la terminal. Aprende a interpretarlo.

ERROR EJEMPLO — SyntaxError
PS C:\TI3011_Python> python ejercicio1.py
File "ejercicio1.py", line 4
return sum(notas) / len(notas
^
SyntaxError: '(' was never closed
📍File "ejercicio1.py", line 4 → El error está en la línea 4
🔴SyntaxError → Tipo de error: error de sintaxis (algo mal escrito)
💬'(' was never closed → Mensaje del error: falta cerrar un paréntesis
Tip: Usa la extensión Error Lens (Módulo 4 → Extensiones) para ver los errores directamente en la línea de código, sin tener que leer la terminal.
