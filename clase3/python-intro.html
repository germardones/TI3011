🐍 Características de Python

Conoce la historia, filosofía y características clave que hacen de Python uno de los lenguajes más poderosos y amados del mundo de la programación.

Historia breve

1989

Guido van Rossum comienza el desarrollo

Buscando un lenguaje de scripting más potente que el intérprete ABC, durante sus vacaciones de Navidad.

1991

Python 0.9.0 — Primera versión pública

Incluye manejo de excepciones, funciones y los tipos de datos centrales: list, dict, str, int.

2000

Python 2.0 — Comprensiones de listas

Se incorpora el recolector de basura y las comprensiones. Crece su adopción en el mundo académico.

2008

Python 3.0 — Rediseño completo

Rompe compatibilidad con Python 2 para eliminar inconsistencias. Unicode nativo, print como función.

Hoy

Python lidera en IA y Data Science

Con librerías como NumPy, Pandas, TensorFlow y PyTorch, domina el machine learning global.

Características clave

Legibilidad

La sintaxis de Python imita al pseudocódigo. Usa indentación obligatoria en lugar de llaves, forzando código limpio y estructurado.

Tipado Dinámico

No necesitas declarar el tipo de una variable. Python lo infiere automáticamente en tiempo de ejecución.

Multiparadigma

Soporta programación orientada a objetos, funcional e imperativa. Tú eliges el estilo según el problema.

Baterías incluidas

La librería estándar cubre desde manejo de archivos hasta servidores web, expresiones regulares y más.

Portabilidad

Corre en Windows, macOS, Linux y más sin modificar el código fuente. Un intérprete por plataforma.

Comunidad masiva

Más de 500.000 paquetes en PyPI, documentación extensa y una comunidad activa que resuelve problemas.

    print("Eres mayor de edad. ")
else:
    print("Eres menor de edad.")
