⌨️ Atajos de Teclado

Los atajos más importantes de VS Code organizados por categoría. Apréndetelos de a poco — empezando por los de uso diario.

Ctrl+Shift+P
Paleta de ComandosAccede a cualquier función de VS Code desde un solo buscador
⭐ Top 1
Ctrl+P
Abrir archivo rápidoEscribe el nombre del archivo para abrirlo sin usar el explorador
⭐ Top 2
Ctrl+S
Guardar archivoGuarda los cambios del archivo activo
Ctrl+Z
DeshacerRevierte el último cambio en el editor
Ctrl+Shift+Z
RehacerVuelve a aplicar el cambio que fue deshecho
Ctrl+F
Buscar en el archivoAbre la barra de búsqueda en el archivo activo
Ctrl+H
Buscar y reemplazarEncuentra texto y lo reemplaza por otro en el archivo
Ctrl+,
Abrir ConfiguraciónAbre el panel de preferencias de VS Code
Alt+↑ / ↓
Mover líneaMueve la línea actual hacia arriba o hacia abajo
Shift+Alt+↓
Duplicar líneaCopia la línea actual justo debajo de ella
Ctrl+D
Seleccionar siguiente coincidenciaSelecciona la siguiente aparición de la palabra seleccionada
Ctrl+/
Comentar / DescomentarActiva o desactiva el comentario en la línea seleccionada
Tab
IndentarAgrega un nivel de indentación (4 espacios por defecto)
Shift+Tab
DesindentarQuita un nivel de indentación de la línea
Ctrl+Shift+K
Eliminar líneaBorra completamente la línea actual
Ctrl+A
Seleccionar todoSelecciona todo el contenido del archivo
Ctrl+Tab
Cambiar entre pestañasNavega entre los archivos abiertos en el editor
Ctrl+W
Cerrar pestañaCierra el archivo activo
Ctrl+G
Ir a líneaNavega directamente a un número de línea específico
Ctrl+B
Mostrar/ocultar SidebarAlterna la visibilidad del panel lateral de exploración
Ctrl+Shift+E
Abrir ExploradorMuestra el panel de exploración de archivos
F12
Ir a definiciónSalta a donde se definió la función o variable bajo el cursor
Ctrl+Shift+F
Buscar en todos los archivosBusca texto en todos los archivos del proyecto
Ctrl+`
Abrir/cerrar TerminalMuestra u oculta el panel de terminal integrado
⭐ Top
Ctrl+Shift+`
Nueva terminalAbre una segunda terminal sin cerrar la actual
↑ / ↓
Historial de comandosNavega por los comandos ejecutados anteriormente en la terminal
Ctrl+C
Interrumpir procesoDetiene el programa Python que está ejecutándose en la terminal
cls / clear
Limpiar terminalcls en Windows, clear en macOS/Linux
F5
Ejecutar con DepuradorInicia la ejecución del script Python con el depurador activado
⭐ Top
Ctrl+F5
Ejecutar sin DepuradorCorre el script directamente en la terminal sin modo debug
F9
Añadir/quitar breakpointEstablece un punto de pausa para inspeccionar el estado del programa
F10
Paso a paso (Step Over)Ejecuta la siguiente línea sin entrar dentro de las funciones
Ctrl+Shift+P → Python: Select Interpreter
Seleccionar intérprete PythonElige qué versión de Python usar para ejecutar tus scripts
Shift+Enter
Ejecutar selección en REPLEnvía el código seleccionado a la terminal Python interactiva
💡

Tip: El botón ▶ (triángulo) que aparece en la esquina superior derecha del editor cuando abres un archivo .py es equivalente a Ctrl+F5. Es la forma más rápida de ejecutar tu script.
