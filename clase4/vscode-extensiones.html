🧩 Extensiones

Las extensiones transforman VS Code en un entorno completo. Aquí están las que necesitas para este curso y las que mejorarán tu experiencia.

Cómo instalar una extensión

🧩
Desde el panel de Extensiones
1
Presiona Ctrl+Shift+X
2
Busca el nombre de la extensión
3
Haz clic en Instalar
4
¡Listo! Se activa automáticamente
También puedes buscar una extensión por su ID exacto en la barra de búsqueda del panel.

🔍 Así se ve la búsqueda de extensiones en VS Code:

🔍 ms-python.python ×

Extensiones esenciales para el curso

🐍
Python
Microsoft
ESENCIAL

La extensión oficial de Python para VS Code. Sin ella no puedes ejecutar ni depurar código Python.

Ejecución de scripts con un clic ▶
Selección del intérprete Python
Formateo y linting automático
Depurador visual integrado
ms-python.python
💡
Pylance
Microsoft
ESENCIAL

Motor de lenguaje avanzado para Python. Potencia el IntelliSense con análisis de tipos estático.

Autocompletado inteligente mejorado
Detección de errores en tiempo real
Documentación al pasar el cursor
Soporte de type hints
ms-python.vscode-pylance

Extensiones recomendadas

🌎
Spanish Language Pack
Microsoft
RECOMENDADO

Traduce toda la interfaz de VS Code al español. Menús, mensajes de error y opciones en tu idioma.

Interfaz completa en español
Sin afectar el rendimiento
MS-CEINTL.vscode-language-pack-es
🎨
Material Icon Theme
Philipp Kief
RECOMENDADO

Reemplaza los íconos de archivos y carpetas por íconos coloridos y claros. Mucho más fácil navegar el explorador.

Íconos para .py, .js, .html y más
Íconos específicos por carpeta
PKief.material-icon-theme
🎯
Error Lens
Alexander
RECOMENDADO

Muestra los errores y advertencias directamente en la línea de código, en lugar de solo en el panel de problemas.

Errores visibles inline
Ideal para aprender
usernamehw.errorlens
🔵
indent-rainbow
oderwat
RECOMENDADO

Colorea los niveles de indentación. Esencial para Python donde la indentación define los bloques de código.

Cada nivel tiene un color
Detecta indentación incorrecta
oderwat.indent-rainbow
