🐍 Programar Python en VS Code

Guía completa paso a paso: desde instalar las extensiones hasta ejecutar y depurar tus scripts de Python.

1
🧩

Instalar la extensión Python

Antes de escribir código Python, necesitas instalar la extensión oficial de Microsoft. Sin ella, VS Code no sabe cómo ejecutar ni entender archivos .py.

Presiona Ctrl+Shift+X para abrir el panel de extensiones, busca Python y haz clic en Instalar. También instala Pylance para mejor autocompletado.

Tip: Después de instalar, VS Code puede pedirte que lo recargues. Haz clic en "Recargar ventana" para activar la extensión.
2
📁

Abrir una carpeta de proyecto

Lo recomendable es trabajar con una carpeta de proyecto en lugar de abrir archivos sueltos. Esto permite que VS Code entienda el contexto de tu código.

Ve a Archivo → Abrir Carpeta (o Ctrl+K → Ctrl+O) y selecciona (o crea) la carpeta donde guardarás tus scripts Python.

Tip: Nombra la carpeta sin espacios ni tildes, por ejemplo mi_proyecto o TI3011. Facilita el trabajo con la terminal.
3
🐍

Seleccionar el intérprete Python

VS Code puede tener múltiples versiones de Python instaladas. Debes indicarle cuál usar. Verás en la Barra de Estado (abajo a la derecha) algo como Python 3.x.x.

Haz clic en ese texto, o usa Ctrl+Shift+P y escribe "Python: Select Interpreter". Selecciona la versión de Python 3 que tienes instalada.

BARRA DE ESTADO — Esquina inferior derecha
🐍 Python 3.12.0 ('base': conda) ← Haz clic aquí para cambiar
¿No ves Python? Primero asegúrate de haber instalado Python desde python.org y marcado la opción "Add Python to PATH" durante la instalación.
4
📄

Crear tu primer archivo .py

En el Explorador (panel izquierdo), haz clic derecho en la carpeta y selecciona "Nuevo Archivo". Nómbralo con extensión .py, por ejemplo hola.py.

Escribe tu código Python. Notarás el resaltado de sintaxis y que el IntelliSense comienza a sugerir completaciones automáticamente.

hola.py 
# Mi primer script Python en VS Code

def saludar(nombre):
    print(f"Hola, {nombre}! Bienvenido a Python.")

mi_nombre = input("¿Cómo te llamas? ")
saludar(mi_nombre)
5
▶️

Ejecutar el script

Tienes tres formas de ejecutar un script Python en VS Code:

A) Botón ▶ — En la esquina superior derecha del editor aparece un triángulo. Haz clic para ejecutar el script actual en la terminal.

B) Clic derecho → "Ejecutar archivo Python en la terminal"

C) Terminal directa — Abre la terminal con Ctrl+` y escribe:

TERMINAL INTEGRADO
PS C:\mi_proyecto> python hola.py
¿Cómo te llamas? Germán
Hola, Germán! Bienvenido a Python.
PS C:\mi_proyecto> █
Tip: Si usas macOS o Linux, el comando es python3 hola.py en lugar de python hola.py.
6
🐛

Depurar errores con el Debugger

Cuando tu programa tiene un error lógico (el código corre pero da resultados incorrectos), el depurador visual es tu mejor herramienta.

Cómo usarlo:

1. Haz clic en el margen izquierdo del número de línea donde quieras pausar → aparece un punto rojo (breakpoint).

2. Presiona F5 para iniciar el depurador.

3. El programa se detendrá en ese punto y podrás ver el valor de cada variable en el panel izquierdo.

4. Usa F10 para avanzar línea a línea y observar cómo cambian los valores.

Tip: También puedes usar print() para mostrar valores intermedios — es la forma más simple de depurar para principiantes.
7
💡

Usar IntelliSense al máximo

IntelliSense es el sistema de autocompletado inteligente de VS Code. Con Pylance instalado, ofrece sugerencias muy precisas.

Trucos útiles:

— Al escribir el nombre de una función y abrir el paréntesis, aparece la firma con los parámetros que acepta.

— Pasa el cursor sobre una función o variable para ver su documentación emergente.

— Presiona Ctrl+Space para forzar las sugerencias en cualquier momento.

— Presiona F12 sobre una función para ir a su definición.
