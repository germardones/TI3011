Las 6 zonas de VS Code

1

Barra de Actividad

La barra vertical más a la izquierda. Cambia entre los paneles principales: Explorador, Búsqueda, Git, Depurador y Extensiones.

Ctrl+Shift+E → Explorador
2

Panel Lateral (Sidebar)

Muestra el contenido del panel activo: archivos de tu proyecto, resultados de búsqueda, cambios de Git o extensiones disponibles.

Ctrl+B → Mostrar/ocultar
3

Editor de Código

Zona central donde escribes. Soporta múltiples pestañas, división en columnas, resaltado de sintaxis e IntelliSense.

Ctrl+P → Abrir archivo
4

Panel Inferior (Terminal)

Terminal integrado donde ejecutas programas, comandos y ves la salida de tus scripts. Puedes abrir múltiples terminales.

Ctrl+` → Abrir terminal
5

Barra de Estado

Barra inferior con información contextual: rama de Git, errores, intérprete de Python seleccionado, posición del cursor y más.

Clic en "Python 3.x" → Cambiar intérprete
6

Paleta de Comandos

La herramienta más potente de VS Code. Ejecuta cualquier acción, configura el editor o instala extensiones desde un único buscador.

Ctrl+Shift+P → Abrir paleta

La Paleta de Comandos

Presiona Ctrl+Shift+P (Windows/Linux) o ⌘+Shift+P (Mac) para abrirla. Es como un buscador de todo lo que puedes hacer en VS Code.
